## 🧠 理论根基

本项目融合了两大理论体系，分别解决减脂中的 **「吃多少」** 和 **「怎么坚持」** 两个核心问题：

### 基础代谢减肥法

> **来源：** 邱超平《减脂生活：基础代谢减肥法》

核心公式：

```
每日饮食总热量 = 基础代谢值（BMR）
```

- **女性 BMR** = 661 + 9.6 × 体重(kg) + 1.72 × 身高(cm) − 4.7 × 年龄
- **男性 BMR** = 67 + 13.73 × 体重(kg) + 5 × 身高(cm) − 6.9 × 年龄

这一方法的核心主张是：**不节食**，保证营养均衡，以基础代谢为热量底线，让身体在不饥饿的状态下持续减脂。

### 环境设计意志力理论

> **来源：** 本杰明·哈迪《意志力陷阱》

核心观点：意志力是有限资源，**真正的改变来自环境设计而不是硬扛**。持续减脂的人并非意志力超群，而是设计了让坚持自动发生的环境。

- 移除清单比意志力管用 100 倍
- If-Then 执行意图是习惯养成的关键工具
- 社会规范的力量大于个人意志

## 🚀 快速开始

### 接入条件

本 Skill 适用于支持自定义 Skill/Prompt 的 AI 应用平台（如 SOLO 等支持自定义指令的系统）。

### 触发方式

用户消息中出现以下任一关键词即可激活：

| 类别 | 关键词 |
|------|--------|
| 减脂诉求 | 想减肥 / 减脂 / 瘦下来 / 控制体重 |
| 饮食询问 | 今天吃什么 / 三餐 / 热量 / BMR / 基础代谢 |
| 意志力崩溃 | 坚持不下去 / 好想吃 / 暴食了 / 想放弃 |
| 明确唤起 | 使用减肥助手 / 启动减脂教练 |

## 🎯 使用示例

### 场景 1：建档与热量目标生成

> **用户：** 帮我减肥  
> **助手：** 先建档 → 输出减脂出发地图  
> **输出：**
> ```
> 📋 您的减脂出发地图
> ─────────────────────────
> 姓名：小陈 | 28岁 | 女
> 身高：165cm | 当前体重：72kg | 目标：55kg
> BMR：1413 千卡/天
> BMI：26.4
> 每日热量目标：1413 千卡
> 预计周期：约 6.8 个月
> ─────────────────────────
> ```

### 场景 2：询问饮食方案

> **用户：** 今天吃什么  
> **助手：** （读取存档 → 调用食谱 → 输出）  
> ```
> 🍳 今日三餐方案（1413 千卡）
> 早餐：水煮蛋1个 + 无糖豆浆250ml + 燕麦30g + 小番茄100g
> 午餐：杂粮饭100g + 鸡胸肉100g + 西蓝花200g
> 晚餐：豆腐100g + 凉拌黄瓜150g
> 进食顺序：汤 → 菜 → 肉 → 饭
> 避忌：汤糖躺烫
> ```

### 场景 3：意志力崩溃 · 环境改造

> **用户：** 坚持不下去了，好想吃炸鸡  
> **助手：**
> ```
> 这种感觉我懂，减脂路上谁都会有这种时刻。
> 您不是意志力差，是环境在跟您作对。
> 哈迪研究了40年发现——真正能持续改变的人，
> 都是设计了让坚持自动发生的环境，而不是靠硬扛。
>
> 现在立刻把厨房里看得见的零食装进不透明袋子
> 放最高柜，或直接扔掉。移除清单比意志力管用100倍。
> ```

### 场景 4：食物判断

> **用户：** 牛油果能吃吗？  
> **助手：** （查食物库 → 输出）  
> ```
> 🟡 限量吃
> 牛油果属于高脂水果，每100g约160千卡，
> 午餐可吃1/4个（约40g），不建议整颗食用。
> ```

## 🛡️ 安全机制

### 病史护栏

当用户填写的 `health_history` 命中以下关键词时，自动加载对应保守方案：

| 关键词 | 处理方案 |
|--------|----------|
| 孕期 / 哺乳期 | 不建议主动减脂；哺乳期需保证营养 |
| 糖尿病 | 强制低 GI 食谱；餐后 30 分钟运动建议 |
| 痛风 | 高蛋白先焯水；每日饮水 ≥2000ml |
| 高血压 | 严格控盐（<5g/天）；不推荐激进断食 |
| 多囊卵巢 / PCOS | 多吃豆制品；建议间歇轻断食 |
| 甲减 / 卵巢早衰 | 严禁过度节食；先就医评估 |
| 胃病 / 肠胃不适 | 避免生冷刺激；少量多餐 |
| 未成年（<18 岁） | 青少年保守方案；发育优先 |

### 急性症状处理

若用户描述出现**胸闷、晕厥、月经停经 >3 个月、严重脱发**等急性症状，**立即建议就医**，不再继续提供减脂方案。

### 核心原则

- ❌ 不诊断、不开药
- ✅ 仅作生活方式指导
- 🏥 涉及医疗问题一律建议就医
- 📉 拒绝激进减脂请求（如"7 天减 10 斤"）
