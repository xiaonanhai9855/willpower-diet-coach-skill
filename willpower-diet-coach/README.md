# 意志力减肥助手 (willpower-diet-coach)

一个把「邱超平·基础代谢减肥法」与「本杰明·哈迪·意志力陷阱」蒸馏成的私人减脂教练 skill。

## 它能做什么

1. 首次对话时帮您建立 Markdown 用户档案（姓名/年龄/性别/身高/体重/目标/病史）
2. 自动计算 BMR、BMI、每日热量目标、预计减脂周期
3. 按基础代谢减肥法生成个性化三餐方案
4. 在您"坚持不下去"时，用《意志力陷阱》的环境设计理论温暖陪伴您
5. 多用户支持：再次唤起时询问"使用旧档案 / 新建档案"
6. 病史安全护栏：糖尿病、痛风、孕期、PCOS、高血压、未成年等自动启用保守方案

## 文件结构

```
willpower-diet-coach/
├── SKILL.md                          ← 主入口（含触发器与所有指令）
├── README.md                         ← 您正在看的这份
├── profiles/                         ← 用户存档（Markdown，每人一个）
│   └── _template.md                  ← 档案模板
└── references/
    ├── fatloss-method.md             ← 基础代谢减肥法完整理论
    ├── meal-plans.md                 ← 男女各一周示例 + 7 款减脂汤 + 热量表
    └── willpower-prompts.md          ← 温暖陪伴型激励话术库
```

## 如何使用

在支持 SKILL.md 的 AI 助手（如 Claude、SOLO、Cursor 等）中：

1. 把整个文件夹放到 AI 助手的 skill 加载路径
2. 跟 AI 说："我想减肥" / "帮我减脂" / "我坚持不下去了"
3. 首次会引导您建档，之后会询问使用哪个档案

## 触发关键词

减肥、减脂、瘦身、控制饮食、想吃东西、坚持不下去、又胖了、节食、热量、三餐、BMR、基础代谢、暴食、想放弃、没动力 ...

## 风格

温暖陪伴型——先接住情绪，再归因到环境，最后给一个具体小动作。绝不喊口号、不堆道理、不评判您。

## 核心信念

> 减脂不是和自己作战，而是和环境共谋。
> 您不需要更强的意志力，您需要一个让坚持自然发生的系统。
