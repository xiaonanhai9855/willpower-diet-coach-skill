# 减脂方法论：基础代谢减肥法

> 来源：邱超平《减脂生活：基础代谢减肥法》读书报告蒸馏

---

## 一、核心原理（一句话）

**每天饮食总热量 = 个人基础代谢值（BMR）**

- 这个量既能保证基本营养与代谢需求，又能自然形成"运动消耗 + 食物热效应"的热量缺口（约 500 千卡/天），让脂肪稳定减少。
- 不需要节食、不需要饿肚子、不会损伤代谢。
- 1 斤脂肪 ≈ 3700 千卡，按上述方法每月稳健减约 5 斤（2.5kg），这就是**科学速度**。

## 二、BMR 计算公式

```
女性 BMR = 661 + 9.6×体重(kg) + 1.72×身高(cm) - 4.7×年龄
男性 BMR = 67  + 13.73×体重(kg) + 5×身高(cm) - 6.9×年龄
```

参考值（成年人通常落点）：
- 女性 1100–1400 千卡
- 男性 1500–1800 千卡

## 三、为什么"节食"不可取（核心反陷阱）

- 长期摄入低于 BMR → 身体启动"低代谢模式" → 基础代谢下降 → 体重越减越难减 → 一旦恢复饮食极易反弹
- 节食还会带来：脱发、月经紊乱、肌肉流失、皮肤变差、情绪暴躁

## 四、三餐搭配总原则

### 黄金三件套
**每一餐都要有：优质蛋白 + 蔬菜 + 适量碳水**

### 三餐能量分配（推荐）
- 早餐 30%（重质量，启动一天代谢）
- 午餐 40%（最丰盛，热量主体）
- 晚餐 30%（清淡，男性可加粗粮，女性以蛋白+蔬菜为主）

### 黄金比例
- 午餐：**主食 : 瘦肉 : 蔬菜 = 1 : 1 : 2**
- 早餐：液体蛋白（豆浆/牛奶）+ 固体蛋白（蛋）+ 粗粮（燕麦/玉米/红薯）+ 蔬菜
- 晚餐：蛋白质（鱼虾/豆腐/鸡胸）+ 大量蔬菜，男性可加半碗粗粮

### 进食顺序（重要！）
**汤 → 菜 → 肉 → 饭**

- 先喝汤垫胃 → 蔬菜膳食纤维占位 → 蛋白质增加饱腹感 → 最后主食，自然就吃得少
- 这是**环境设计**思维：不是控制食欲，是设计顺序让食欲被自然降低

### 四大避忌（口诀：汤糖躺烫）
- **汤**：浓汤、勾芡汤、骨头汤（高油高脂）
- **糖**：含糖饮料、奶茶、糕点、果干
- **躺**：饭后立即躺/坐 1 小时以上
- **烫**：>65℃ 的热饮热食（伤食道，也容易吃过量）

## 五、食物分类速查

### ✅ 推荐（可放心吃）

| 类别 | 推荐 |
|---|---|
| 优质蛋白 | 鱼、虾、鸡胸、瘦牛肉、鸡蛋、低脂牛奶、无糖酸奶、豆腐、豆浆 |
| 优质碳水 | 燕麦、糙米、玉米、红薯、紫薯、藜麦、全麦面包、杂粮饭 |
| 蔬菜 | 绿叶菜（菠菜、油菜、生菜、芹菜）、瓜类（冬瓜、黄瓜、丝瓜）、菌菇、西兰花、番茄 |
| 水果 | 苹果、蓝莓、草莓、柚子、猕猴桃、火龙果（每天 ≤ 200g） |
| 调味 | 醋、柠檬、蒜、姜、葱、低钠盐、黑胡椒 |

### ⚠️ 限量（每周 1–2 次小份）

牛油果、坚果（每天 1 小把 ≈ 10g）、奶酪、香蕉、葡萄、芒果、糙米饭加量、全麦意面

### ❌ 不推荐

| 类别 | 理由 |
|---|---|
| 油炸食品 | 炸鸡、薯条、油条、天妇罗——热量爆表 |
| 含糖饮料 | 奶茶、可乐、果汁、运动饮料——液体糖最易胖 |
| 糕点甜品 | 蛋糕、面包、饼干、巧克力——精制糖+反式脂肪 |
| 加工肉 | 香肠、培根、火腿、午餐肉——高钠 + 添加剂 |
| 精制主食 | 白面条、白馒头、白米饭过量、糯米食品 |
| 高 GI 水果 | 荔枝、龙眼、榴莲、菠萝蜜、果干 |

## 六、运动建议（锦上添花，不是主菜）

- 减脂 = **80% 饮食 + 20% 运动**
- 推荐：每天快走/慢跑/游泳 30–45 分钟 + 每周 2 次力量训练（保肌肉）
- 餐后 30 分钟散步 15 分钟 → 帮助血糖平稳、减少脂肪合成
- **不建议**：空腹长时间剧烈运动（易低血糖+肌肉流失）

## 七、特殊场景

### 平台期（连续 2 周不掉秤）
1. 检查是否真的吃到 BMR（很多人不知不觉吃多）
2. 调整三餐结构（如把午餐主食减半，蛋白加量）
3. 加一次力量训练
4. 一顿"欺骗餐"刺激代谢（每周 1 次，热量 ≤ 当日 BMR×1.2）

### 经期
- 经期前 1 周：碳水适当增加（红薯、燕麦），缓解情绪
- 经期中：不强制运动，保证睡眠
- 经期后 1 周：减脂黄金期，全面执行计划

### 出差 / 聚餐
- 提前看菜单，心里有数
- 进食顺序"汤→菜→肉→饭"照做
- 酒精尽量推掉，必须喝时优选红酒一小杯
- 第二天不补偿性少吃，按计划继续

## 八、衡量进步的尺子

- 体重（每周同一时间、同一状态测 1 次）
- 围度（腰围、臀围、大腿——比体重更诚实）
- 体脂率（家用秤估算即可）
- 镜子里的自己
- 衣服的松紧

**重要**：不要每天称重，会被水分波动影响心情。

## 九、核心信念

> 减脂不是惩罚自己，是好好爱自己。
> 你不是在变瘦，你是在变得更有掌控感。
