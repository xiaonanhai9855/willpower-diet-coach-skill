# 一周三餐示例 & 热量速查

> 说明：以下示例按 **女性 BMR ~1200 千卡 / 男性 BMR ~1600 千卡** 设计，实际使用时按用户 BMR 等比例缩放（如女性 BMR=1300，则把每项 ×1.08）。

---

## 一、女性减脂一周食谱（约 1200 千卡/天）

### 周一
- **早餐**（300 kcal）：无糖豆浆 250ml + 水煮蛋 1 个 + 燕麦片 30g + 圣女果 100g
- **午餐**（450 kcal）：杂粮饭 80g + 清蒸鲈鱼 100g + 蒜蓉西兰花 200g + 紫菜蛋花汤 1 碗
- **晚餐**（350 kcal）：豆腐虾仁汤 + 凉拌菠菜 200g + 黄瓜 1 根

### 周二
- **早餐**：低脂牛奶 250ml + 全麦面包 1 片 + 煎蛋（少油）+ 苹果半个
- **午餐**：糙米饭 80g + 鸡胸肉煎 100g + 清炒油麦菜 200g + 冬瓜汤
- **晚餐**：蒸鸡蛋羹（1 个蛋 + 虾仁 50g）+ 焯西蓝花 150g

### 周三
- **早餐**：无糖酸奶 150g + 蓝莓 50g + 玉米半根 + 水煮蛋
- **午餐**：荞麦面（干重 60g）+ 牛肉 80g + 大量青菜 + 番茄
- **晚餐**：清蒸三文鱼 100g + 凉拌秋葵 + 紫薯半个

### 周四
- **早餐**：豆浆 250ml + 茶叶蛋 + 蒸红薯 100g + 黄瓜
- **午餐**：杂粮饭 80g + 清炒虾仁 + 蒜蓉空心菜 + 海带汤
- **晚餐**：豆腐青菜锅（北豆腐 150g + 各种青菜）

### 周五
- **早餐**：燕麦粥（燕麦 30g 煮制）+ 水煮蛋 + 火龙果 100g
- **午餐**：藜麦饭 80g + 香煎鸡腿（去皮）80g + 上汤娃娃菜
- **晚餐**：番茄豆腐汤 + 清炒生菜 + 鸡蛋 1 个

### 周六
- **早餐**：无糖豆浆 + 全麦三明治（生菜+番茄+鸡胸）
- **午餐**：糙米饭 80g + 红烧鸡腿（去皮，少油少糖）+ 蒜蓉菠菜
- **晚餐**：蔬菜豆腐汤 + 凉拌木耳 + 水煮蛋

### 周日（弹性日，可选 1 顿欺骗餐）
- **早餐**：燕麦牛奶 + 水煮蛋 + 苹果
- **午餐（欺骗餐）**：自由选择，但热量 ≤ 800 kcal
- **晚餐**：清淡蔬菜汤 + 蛋白质（鱼/虾/鸡胸 100g）

## 二、男性减脂一周食谱（约 1600 千卡/天）

### 周一
- **早餐**（400 kcal）：豆浆 300ml + 水煮蛋 2 个 + 燕麦片 50g + 蒸玉米半根
- **午餐**（600 kcal）：杂粮饭 120g + 清蒸鲈鱼 150g + 蒜蓉西兰花 250g + 番茄蛋花汤
- **晚餐**（450 kcal）：糙米饭 60g + 鸡胸肉煎 100g + 大量青菜 + 紫菜汤

### 周二
- **早餐**：低脂牛奶 300ml + 全麦面包 2 片 + 煎蛋 2 个 + 苹果
- **午餐**：藜麦饭 120g + 牛肉炖萝卜 150g + 清炒油麦菜
- **晚餐**：杂粮饭 60g + 清蒸虾 150g + 焯西蓝花

### 周三
- **早餐**：无糖酸奶 200g + 蓝莓 + 玉米 + 茶叶蛋 2 个
- **午餐**：荞麦面（干重 90g）+ 牛肉片 120g + 青菜
- **晚餐**：糙米饭 60g + 清蒸三文鱼 120g + 凉拌秋葵

### 周四
- **早餐**：豆浆 + 蒸红薯 150g + 水煮蛋 2 个
- **午餐**：杂粮饭 120g + 清炒虾仁 150g + 蒜蓉空心菜 + 海带汤
- **晚餐**：糙米饭 60g + 香煎鸡胸 100g + 焯青菜

### 周五
- **早餐**：燕麦粥（燕麦 50g）+ 水煮蛋 2 个 + 火龙果
- **午餐**：藜麦饭 120g + 香煎鸡腿（去皮）120g + 上汤娃娃菜
- **晚餐**：番茄豆腐汤 + 清炒生菜 + 鸡蛋 2 个 + 半碗杂粮饭

### 周六
- **早餐**：豆浆 + 全麦三明治（双层鸡胸+生菜+番茄）
- **午餐**：糙米饭 120g + 牛排（瘦）150g + 蒜蓉菠菜
- **晚餐**：蔬菜豆腐锅 + 凉拌木耳 + 水煮蛋

### 周日（弹性日）
- **早餐**：燕麦牛奶 + 水煮蛋
- **午餐（欺骗餐）**：自由选择，热量 ≤ 1000 kcal
- **晚餐**：清淡蔬菜汤 + 蛋白质（150g）+ 半碗杂粮饭

## 三、7 款减脂汤品（餐前喝，垫胃增饱腹）

1. **冬瓜紫菜汤**：冬瓜 200g + 紫菜 5g + 一点姜片，少油少盐
2. **番茄豆腐汤**：番茄 1 个 + 北豆腐 100g + 鸡蛋 1 个
3. **海带排骨汤**（排骨先焯水去油）：海带 100g + 瘦排骨 80g
4. **菌菇蛋花汤**：金针菇 + 香菇 + 鸡蛋
5. **白菜豆腐汤**：白菜 200g + 嫩豆腐 100g
6. **丝瓜蛋汤**：丝瓜 1 根 + 鸡蛋 1 个
7. **酸辣木耳汤**：黑木耳 + 番茄 + 一点醋（开胃促代谢）

> 所有汤品控制在 100 kcal 内，餐前 5 分钟喝。
